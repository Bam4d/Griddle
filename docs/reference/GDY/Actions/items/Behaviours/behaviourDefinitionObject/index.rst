Behaviour Objects
=================

:Description: The object or list of objects that this behaviour applies to.

:Possible Values:

.. list-table::

   * - **Value**
     - **Type**
     - **Description**
   * - ``[string]``
     - string
     - A single object the behaviour is applied to.
   * - List of Objects
     - array
     - A list of object the behaviour is applied to
