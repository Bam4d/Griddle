TileSize
========

:Description: The common size of all the tiles in the environment. If tile images are supplied with different sizes to this, they will be scaled to this size

.. list-table::

   * - **Data Type**
     - **YAML Key**
     - **Default Value**
   * - integer
     - ``TileSize``
     - ``10``
